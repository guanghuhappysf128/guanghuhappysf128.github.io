# Personal Academic Website

A professional academic portfolio website showcasing research, teaching, and publications.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Sections**:
  - Home/Hero section with call-to-action
  - About section with bio
  - Research interests and current projects
  - Teaching experience and courses
  - Publications
  - Contact information

## Customization

To personalize the website, update the following:

1. **index.html**:
   - Replace "Your Name" with your actual name
   - Update the email and office location in the contact section
   - Add your social media links (Google Scholar, GitHub, LinkedIn)
   - Modify research projects and publications based on your actual work
   - Update course information with your teaching experience

2. **styles.css**:
   - Customize colors in the `:root` variables section
   - Adjust fonts if desired
   - Modify spacing and layout as needed

3. **CV.pdf**:
   - Ensure your CV PDF is in the homepage folder
   - The "Download CV" button links to this file

## File Structure

```
homepage/
├── index.html       # Main HTML file
├── styles.css       # CSS styling
├── CV.pdf          # Your CV (already present)
└── README.md       # This file
```

## Viewing the Website

### Local Development
Simply open `index.html` in a web browser to view the site locally.

### Deployment Options

1. **GitHub Pages**: 
   - Push to a GitHub repository
   - Enable GitHub Pages in repository settings

2. **Netlify/Vercel**: 
   - Drag and drop the folder or connect to GitHub

3. **University Server**: 
   - Upload files to your university web space
